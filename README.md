# UX/UI Portfolio Template

## Project Objective
The objective of this project is to design and develop a professional portfolio template for a UX/UI Designer specialized in mobile and web applications.

The portfolio focuses on presenting not only final visual results,
but also the full design and development process, including planning, wireframes, UI design, prototyping, and frontend implementation.

## Academic and Professional Approach
This project follows an academic and professional methodology,
documenting every phase of the process and using version control
to track progress and decisions.
